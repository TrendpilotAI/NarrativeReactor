# COMMAND REFERENCE

## Quick Commands

| Command | Action |
|---------|--------|
| `episode [X.X]` | Load specific episode context |
| `character [name]` | Load character profile |
| `colors` | Display brand color palette |
| `messaging` | Show messaging hierarchy |
| `generate [platform] for [episode]` | Create platform content |
| `visual prompt for [scene/character]` | Generate AI image prompt |
| `full package for [episode]` | Generate complete content package |
| `compare [A] vs [B]` | Compare two content options |
| `refine` | Iterate on last output |
| `brand check` | Verify brand compliance |

## Platform Shortcuts

| Shortcut | Expands To |
|----------|------------|
| `tw` | Twitter/X |
| `li` | LinkedIn |
| `th` | Threads |
| `all` | All platforms |

# INTEGRATION WITH GOOGLE ANTIGRAVITY

## Using Narrative Reactor in Antigravity

### Manager View Workflow

1. **Open Antigravity** ‚Üí Launch the IDE
2. **Create Workspace** ‚Üí Name: "narrative-reactor-content"
3. **Load Knowledge Base** ‚Üí Import Story Bible markdown files
4. **Start Agent** ‚Üí Describe content needs in natural language
5. **Review Artifacts** ‚Üí Check generated content packages
6. **Provide Feedback** ‚Üí Leave comments on artifacts
7. **Export** ‚Üí Send to scheduling tools

### Editor View Workflow

For direct prompt engineering and code editing:

```typescript
// In Antigravity Editor View

// Quick content generation

  prompt: 'Episode 3.1 Twitter copy with Maya focus',
});

// Run in Genkit Developer UI
// genkit start -- npx tsx src/generate.ts
```
