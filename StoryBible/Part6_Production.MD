# CHAPTER 1: PLATFORM SPECIFICATIONS

## X (TWITTER)

| Element | Specification |
|---------|--------------|
| **Video** | 1920x1080 (16:9) or 1080x1080 (1:1) |
| **Max Duration** | 2:20 |
| **Format** | MP4, H.264 codec |
| **Image** | 1600x900 or 1200x1200 |
| **Image Format** | PNG or JPG, max 5MB |
| **Copy Length** | 280 characters max |
| **Thread Limit** | 25 tweets per thread |
| **Hashtag Strategy** | 2-3 max per post |

### Best Practices
- First 3 seconds must capture attention
- Captions essential (85% watch muted)
- Use threads for longer narratives
- Quote-tweet format for engagement

---

## LINKEDIN

| Element | Specification |
|---------|--------------|
| **Video** | 1920x1080 (16:9) |
| **Max Duration** | 10 minutes |
| **Format** | MP4 |
| **Image** | 1200x627 or 1080x1080 |
| **Image Format** | PNG or JPG |
| **Copy Length** | 3000 characters |
| **Visible Before Fold** | First 140 characters |

### Best Practices
- Hook must appear in first 140 characters
- Document carousels for educational content
- Tag relevant companies/people strategically
- Post during business hours (8-10 AM, 12-2 PM EST)

---

## THREADS

| Element | Specification |
|---------|--------------|
| **Video** | 1080x1920 (9:16 preferred) or 1080x1080 |
| **Max Duration** | 5 minutes |
| **Image** | 1080x1350 or 1080x1080 |
| **Image Format** | PNG or JPG |
| **Copy Length** | 500 characters |
| **Tone** | Conversational, authentic |

### Best Practices
- More casual tone than LinkedIn
- Behind-the-scenes content performs well
- Community interaction expected
- Less formal hashtag usage

# CHAPTER 5: TYPOGRAPHY SPECIFICATIONS

## FONT STACK

| Use Case | Font | Weight | Size Range |
|----------|------|--------|------------|
| Headlines | Helvetica Neue | Bold | 48-72pt |
| Subheads | Helvetica Neue | Medium | 28-36pt |
| Body Copy | Georgia | Regular | 18-24pt |
| Quotes | Georgia | Italic | 24-32pt |
| Data/Technical | SF Mono / Courier | Regular | 16-20pt |
| UI Elements | SF Pro / System | Various | 14-18pt |
| Captions | Helvetica Neue | Light | 14-16pt |

## TYPOGRAPHY RULES

1. **Headlines:** Always Helvetica Neue Bold, sentence case
2. **Body text:** Georgia for warmth and readability
3. **Technical/Data:** Monospace for authenticity
4. **Maximum 2 fonts per piece** (headline + body)
5. **Line height:** 1.4-1.6 for body text

# CHAPTER 9: SOUND DESIGN SPECIFICATIONS

## SIGNATURE SOUNDS

### Signal Studio Activation Tone
- Clear, resonant chime (C major)
- 1.5 seconds duration
- Clean, digital but warm
- Should feel like "possibility realized"

### Processing Sound
- Subtle, rhythmic pulse
- Layered frequencies suggesting multiple agents
- Builds subtly, resolves cleanly
- Duration: 11 seconds (matches the metric)

### Transition Sound
- Swoosh with digital undertone
- Used between scenes
- 0.5-1 second duration

## MUSIC GUIDELINES

| Moment Type | Music Style | Tempo | Key |
|-------------|-------------|-------|-----|
| Problem state | Minimal, tense | 60-70 BPM | Minor |
| Transition | Building, hopeful | 80-90 BPM | Transitional |
| Solution state | Warm, confident | 90-110 BPM | Major |
| Celebration | Triumphant, full | 110-120 BPM | Major |
