    Antigravity Critical Patterns

    Required Reading for All Agents

    This document contains critical patterns learned from repeated mistakes. Every agent MUST consult this before generating code.

    How Patterns Are Added

    A pattern is promoted to this document when: 1. The same mistake has occurred 3+ times 2. The solution is non-obvious but must be followed every time 3. It represents a foundational requirement
